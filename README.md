# MemeMe
MemeMe is a memeÂ­ generating app that enables a user to attach texts to a picture that they take or already on their phones. After adding text to an image chosen from the Photo Album or Camera, the user can share it with friends. MemeMe also temporarily stores sent memes which users can browse in a table or a grid.

### First View
First view is to view the memes that are already created. If it is empty, the user can create a new meme simply tapping + button the the top right.

### Create Meme
The User can select a photo from their phones or they can take a new photo. They can edit the text and save the photo. They can also share the image in facebook twitter, send as a message, etc...
